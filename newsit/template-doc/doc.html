Setting up Dev Tools

Unzip the theme package to any folder and open a command line at that location.

shopapp's dev tools require NodeJs and the Gulp CLI package. Assuming you have Node and Git installed, the following command installs Gulp Command Line Utility globally so the gulp command can be used anywhere in your system.

Node v18.16.0

NPM v9.2.0

npm install gulp-cli -g

When you have installed Gulp, run npm install and the required packages for the shopapp dev tools will be downloaded to the node_modules directory.

npm install

If you see errors such as EACCESS during installation, please follow these instructions at npm docs to install Node via nvm which resolves the permissions errors.

Run gulp in the shopapp/ folder to serve the project files using BrowserSync. Running gulp will compile the theme and open /index.html in your main browser.

gulp

While the gulp command is running, files in the assets/scss/, assets/js/ and html/ folders will be monitored for changes. Files from the assets/scss/ folder will generate injected CSS.

Hit CTRL+C to terminate the gulp command. This will stop the local server from running.

Theme without Sass, Gulp or Npm

If you’d like to get a version of our theme without Sass, Gulp or Npm, we’ve got you covered. Run the following command:

gulp build:dev

This will generate a folder dev which will have unminified CSS, Html and Javascript.

Minified version

If you’d like to compile the code and get a minified version just run the following Gulp command:

gulp build:dist

This will generate a folder dist which will have minified CSS, Html and Javascript.

We have already compiled dev and dist folder just for the reference. if you want to update color or any other code you should be work in shopapp folder and then compile again.

All Header, Footer, Head and script common code are in partials folder.

Edit the Color Scheme

Using SCSS you can easily transform the color scheme of your site.

Editing the default color scheme variables

1. Open the _variables.scss file

The _variables.scss file is located in the /assets/scss/core/custom directory of the theme package.

2. Change variable values

Inside _variables.scss under the Color scheme section you'll see the color variables. Modify the HEX values to change the color scheme.

$px-primary: #008aff !default;
